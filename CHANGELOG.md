# 更新日志

本文档记录 Chaoxing PDF Dumper 的用户可见变化。发布到 Greasy Fork 或 ScriptCat 时，可将对应版本下的要点复制到发布页的“更新日志”中。

## 0.6.1

- 优化课件下载面板的扫描、选择和下载体验。
- 保留课程页扫描和当前章节扫描能力。
- 支持显示非 PDF 资源状态，方便确认章节内的全部资源。
- 支持复制单条或全部接口 JSON，方便排查资源信息。
- 保留批量下载取消、下载地址 fallback、剪贴板 fallback 和 SPA URL 监听。

## 维护说明

- 功能更新、兼容性修复和用户可见行为变化，都应在这里补充。
- 仅文档、截图、注释等不影响脚本运行的改动，可以不提升 userscript 版本。
- Greasy Fork / ScriptCat 发布页的更新日志建议保持简短，写 1 到 4 条用户能看懂的变化。
